About Me


Hello everyone! My name is Kanishk, and I am a 2023 B.Tech graduate in Information Technology from Amity University. I am currently working as a TechOps Engineer at Zopper.

This is my first website built with Django. It serves as an introduction to Django, where I've learned about URLs, templates, views, and how to redirect from one endpoint to another using urls.py. I also learned how to create functions in views.py to render templates. The website includes a feature where users can input text, and it will remove punctuation, capitalize letters, eliminate extra spaces and lines, and count the number of characters in the text.
